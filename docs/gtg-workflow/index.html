GTG workflow

To make best use of the GTG platform for developing Galaxy tools, following the workflow below is recommended.

  1. Launch a GTG platform
  2. Launch a Galaxy instance
  3. Create XMLs
  4. Edit XML components
  5. Build Tool Repository
  6. Add non-XMLs to Tool Repository
  7. Publish tool repository to Test Tool Shed
  8. Install tool from Test Tool Shed in Galaxy
  9. Test tool
