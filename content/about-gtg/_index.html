Galaxy Tool Generator (GTG) is a platform which allows devloping Galaxy tool through web interfaces.

GTG components

The GTG platform consists of three components:

  • two Drupal modules: galaxy_tool_generator and galaxy_tool_generator_ui.
    • the galaxy_tool_generator module maps each Galaxy tool config XML elements to a drupal webform components. The “Type” column in the figure table below is an example of showing the mapping between XML elements and webform components.
    • the galaxy_tool_generator_ui module builds user interfaces

  • gtgdocker: a docker image that wraps a drupal site with the two GTG modules installed.

  • gtgr: an R package that launches, stops and deletes the GTG platform.

Requirements

The GTG platform runs as a Docker container and is controled by the R package gtgr. Therefore, Docker and R are required.
